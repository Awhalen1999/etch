# etch-cli

a text horror game. the only way is down.

```
npx etch-cli
```

the surface is dead. you fell. enemies live below. the queen waits at
depth 200.

requires node 18+. prebuilt binaries provided for:

- macOS (arm64 + x64)
- linux (x64 + arm64)
- windows (x64)

your save lives in `~/.etch/`. nothing leaves your machine except the
inscriptions you carve.
